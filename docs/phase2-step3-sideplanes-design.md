## 1. 形態の読み取りと物理像

- ML66 §3.8: 側面系は -20°C 以深の「粉雪」を構成。Fig.1 の S1 スケッチは、**共通の線(スパイン)から複数の薄板がフィン状に異なる角度で張り出す**集合体。S2 は小さな鱗状板の重なり。
- 物理像: 凍結雲粒起源の**多結晶**(ロゼットと同族)。各成分は基底面が卓越した薄板で、成分間の c 軸方位が異なる。

## 2. 結晶学アンカー: CSL 双晶角 70.3°/[112̄0]【文献確定済み 2026-06-10】

- 多結晶雪結晶の成分間 c 軸角が約 70° に集中する現象は、Kobayashi, Furukawa, Takahashi & Uyeda (1976) "Cubic structure models at the junctions in polycrystalline snow crystals", *J. Crystal Growth* **35**, 262–268 が、接合部に立方晶構造を持つ双晶モデルで説明した(空間型の板状集合で頻出)。
- CSL(一致格子)理論の系譜(Kobayashi & Furukawa ら)では、低エネルギー境界として **70.3°/[112̄0]**(a 軸まわり 70.3° 回転)が導かれる。−20°C 以深で観測される交差板結晶は 2〜4 枚の薄い基底面板の集合で、各板がこの 70.3°/[112̄0] の CSL 粒界に沿って延びると報告されている — 本設計の「スパイン = a 軸」はこの関係の直接の写像。
- 幾何への翻訳: 各フィンのスパイン辺を **a 軸〈11-20〉方向**に取ると(§3)、スパインは両フィンの基底面内にあり、c 軸はともにスパインに直交。したがって**フィン間の二面角 = c 軸間角**となり、70.3° がそのまま描画パラメータに写る。
- 採用定数: **CSL_TWIN_ANGLE_DEG = 70.3**。JSDoc に上記出典と、立方晶接合モデルの四面体角 acos(1/3) ≈ 70.53° との関係(理想立方双晶の角。CSL の近一致計算では 70.3°)を注記する。フィン本数 4〜7 は観測の「2〜4 枚の板」(≒半フィン 4〜8 枚)と整合。

## 3. 部品: 半六角フィン

- アウトライン: 正六角形を**長対角線(対頂点間)で半裁**した四角形 `halfHexOutline(R)`。
  - スパイン辺 = 長対角線(長さ 2R)= 頂点方向 = **a 軸**(A_AXES 規約と整合)。
  - 外周 3 辺 = {10-1̄0} 柱面トレース。内角は **60° / 120° / 120° / 60°**。
- 押し出し厚 t = 0.05·R(薄板)。原点 = スパイン辺の中点、スパイン = ローカル +Y、フィンは +X 側へ張り出す(回転は二面角で与える)。
- マテリアル: COLORS.wing / flatShading: true(統一規約)。

## 4. アセンブリ: createSidePlanes(rng)

- フィン本数 N ∈ [4, 7](シード乱数・一様)。
- 二面角スキーム(検収論点 2 で選択):
  - **案 B(推奨)**: 基準角 θ0(シード乱数)+ 各フィンに {0, ±70.3°, ±109.7°, ±140.6°} から重複なく選んだオフセット + ジッタ ±6°。最小角間隔 20°(mod 360°)を満たさない場合は再抽選、上限到達で案 A 相当の等間隔へフォールバック。(±140.6° の両符号化は N=7 と集合要素数の不整合解消、2026-06-10 検収時修正)
  - 案 A: ロゼットと同式(シード乱数 + 最小間隔制約のみ。結晶学アンカーなし)。
- サイズジッタ: 各フィン R_p = R_base × (0.8〜1.2、シード乱数)。
- スパイン方向スタッガ: 各フィンのスパイン中点を、スパイン軸上のスパン ±0.4·R_base 内でシード乱数オフセット(Fig.1 のずれた重なりを再現)。
- 中心構造: 追加ジオメトリなし。薄板同士がスパイン近傍で自然に交差して芯が立つ(ロゼットの apex 埋め込みと同思想)。
- 決定性: seed → 本数・角度列・サイズ列・オフセット列がすべて再現(既存 mulberry32 経路、針・ロゼットと同流儀)。

## 5. 寸法方針

- 全径(フィン張り出しの差し渡し ≈ 2×(R_base·√3/2 + 厚み))を、ロゼット(≈2.75)と同級〜やや小さめに収める。R_base と t の最終値は実装時に既存形態と実測比較して確定(砲弾と同じく一任、採用値と根拠を報告)。
